COSC 434 – Advanced Web Programming: Protecting the Recipe App Using Middleware
In this lab, I implemented middleware on some CRUD operations(create, edit, update, delete) in order to limit the accessibility of the user when they don't log in. 
It's done by creating a middleware file, then giving it the correct logic to indicate the message that should appear. Then, I defined the routes for the middleware protection.
Finally, everything was tested and functioned correctly.
